Data Transmission

To send authorization codes or login credentials to the server, you must construct a Form. This is the interactive bridge between the user and the mainframe.

1. The Form Wrapper

Everything interactive belongs inside the <form> container. By itself, it is invisible, but it groups your inputs together.

2. Inputs & Buttons

The <input> tag creates a field for the user to type in. It is a self-closing void tag. You use the `type` attribute to define its behavior (e.g., `type="text"`, `type="password"`).
Finally, the <button> tag creates a clickable trigger to submit the form data.

<form>
  <input type="text" placeholder="Username">
  <button>TRANSMIT</button>
</form>
