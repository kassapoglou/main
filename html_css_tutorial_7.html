Structuring Telemetry

To parse incoming raw data (like ship statuses, resource counts, or crew rosters), we must organize it into grids. In HTML, this is called a Table.

1. The Matrix and Rows

The entire structure is wrapped in the <table> tag. Inside, we slice the table horizontally into rows using the <tr> (Table Row) tag.

2. Headers vs. Data

Inside each row, you place individual cells. There are two types of cells:

  • <th> (Table Header): Used in the first row to define columns. It makes text bold and centered.
  • <td> (Table Data): Used in all subsequent rows to hold the actual information.
<table>
  <tr>
    <th>Ship Class</th>
  </tr>
  <tr>
    <td>Fighter</td>
  </tr>
</table>
