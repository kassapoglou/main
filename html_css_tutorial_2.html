Connecting the Galaxy

A single page is useless in the vastness of the web. To jump from one sector (page) to another, we use the Anchor tag: <a>.

1. The Anchor Tag and Href

The <a> tag requires an attribute to work. Attributes go inside the opening tag to provide extra instructions. For links, we use href (Hypertext Reference) to specify the destination URL.

<!-- Linking to another page -->
<a href="armory.html">Enter Armory</a>
> RENDER OUTPUT: Enter Armory

2. External Jumps

You can also route completely outside of your local network by providing a full web address (like `https://google.com`).

<a href="https://nasa.gov">Earth Command</a>
