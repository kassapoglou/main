External Toolboxes

Python is incredibly powerful, but it doesn't load every tool by default to save memory. A Module is a separate file containing pre-written code (toolboxes) created by other programmers. We bring them into our script using the import keyword.

1. Importing a Module

Put your imports at the very top of your script. This tells the computer to fetch the external toolbox before doing anything else.

# Fetch the math module
import math

# Fetch the random module
import random

2. Using the Module's Functions

To use a function from a module, you type the module's name, a dot (.), and then the function name. This is called "dot notation".

import math

# Ask the math module to calculate the square root
result = math.sqrt(16)
print(result)
