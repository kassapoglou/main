Reacting with Watchers

Sometimes you need to perform "side effects" when data changes—like making an API call, triggering an animation, or automatically firing an EMP when enemy counts get too high. For this, we use watchers.

1. The watch Option

Inside the watch object, create a function with the exact same name as the data property you want to observe. Vue automatically passes the new value and the old value to this function.

watch: {
  // Watch the 'hullIntegrity' data variable
  hullIntegrity(newValue, oldValue) {
    if (newValue < 20) {
      this.triggerAlarm();
    }
  }
}

2. Computed vs. Watch

Use computed when you want to derive a new value to display on the screen. Use watch when you want to do something (execute a method) in response to a change.
