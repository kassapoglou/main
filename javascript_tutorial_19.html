The Document Object Model

JavaScript doesn't just run math in the background. It can talk directly to the HTML on your screen. The browser turns your HTML tags into a massive family tree called the DOM.

1. Selecting an Element

To hack into a specific piece of the UI, you need its `id`. We use the document.getElementById() method to grab it.

<div id="comms">Offline</div>

let node = document.getElementById("comms");

2. Modifying the Element

Once you have grabbed the element, you can change its text using .innerText, or change its CSS styles using .style.

node.innerText = "Online!"; // Changes text
node.style.color = "lime"; // Changes font color
node.style.backgroundColor = "black";
