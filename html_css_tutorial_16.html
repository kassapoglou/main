Pseudo-Classes

A static interface is a dead interface. To provide visual feedback when a user interacts with elements (like buttons or links), we use Pseudo-classes. These are keywords added to a selector that define a special "state".

1. The Hover State (:hover)

Triggers when the user's cursor rests over an element. It is essential for letting users know an element is interactive.

.btn:hover {
  background: cyan;
}
Simulated Hover

2. The Active State (:active)

Triggers the exact moment an element is being clicked (mouse button held down). It provides the tactile sensation of pushing a physical button.

.btn:active {
  transform: scale(0.9);
}
Simulated Click
