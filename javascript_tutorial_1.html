The Developer Console

Before manipulating the screen, you must learn to talk to the engine. The console is an object that provides access to the browser's debugging console. Your users won't see this—it is strictly for developer telemetry.

1. Standard Output

The primary method to print messages or evaluate variables. Use quotes for text (strings).

console.log("Systems Online");

2. Warnings

Outputs a warning message. In a real browser, this appears yellow with a warning icon.

console.warn("Shield capacity dropping.");

3. Critical Errors

Outputs an error message. It appears bright red and usually prints a stack trace to help you find bugs.

console.error("HULL BREACH DETECTED!");
