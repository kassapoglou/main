Binding HTML Attributes

Mustaches {{ }} only work inside text content. If you want to dynamically update an HTML attribute (like id, class, src, or disabled), you must use the v-bind directive.

1. The v-bind Syntax

By adding v-bind: in front of an attribute, Vue knows to evaluate the quotes as JavaScript, looking for your data variables.

<!-- Binds the image source to 'pilotAvatar' data -->
<img v-bind:src="pilotAvatar">

2. The Shorthand (:)

Because v-bind is used so frequently, Vue provides a crucial shorthand. Just use a colon : before the attribute.

<!-- These do the exact same thing! -->
<div v-bind:class="shieldState"></div>
<div :class="shieldState"></div>
