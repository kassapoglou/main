Classes and Objects

Object-Oriented Programming (OOP) allows us to model real-world concepts. A Class is a blueprint. An Object is the actual thing built from that blueprint.

1. The Blueprint (Class)

We define a class using the class keyword. The special __init__ function sets up the object's initial attributes (like setting a robot's health when it is built).

class Starship:
    def __init__(self, name):
        self.name = name
        self.shields = 100

    def status(self):
        print(f"{self.name} shields at {self.shields}%")

2. Instantiation (Creating the Object)

To build an object, you call the Class name like a function. You can then use dot notation to trigger its built-in actions (Methods).

# Instantiate the object
my_ship = Starship("Apollo")

# Call its method
my_ship.status()
