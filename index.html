Ξεκίνα να προγραμματίζεις από σήμερα

Υπάρχει ο φόβος και η λανθασμένη άποψη ότι ο προγραμματισμός είναι δύσκολος, και ότι δεν είναι κάτι προσιτό για όλους. Λάθος! Αν κάποιος καταλαβαίνει τα μαθήματα στο site μου, τότε καταλαβαίνει προγραμματισμό. “Είναι τόσο απλά?” θα αναρωτηθεί κάποιος. Ναι είναι. Απλά πρέπει να μπουν τα κουτάκια της γνώσης με την σωστή σειρά και την κατάλληλη στιγμή για να μπορέσει ο εκπαιδευόμενος να δει την αλληλουχία ανάμεσα τους. Και αυτό προσπαθώ να κάνω όσο το δυνατόν μπορώ μέσα από το site.

ΑΠΟ ΠΟΥ ΞΕΚΙΝΑΩ?

Ακολούθησε ένα από τα career paths (σύντομα διαθέσιμο). Μπορείς να επιλέξεις backend development, frondend development, ή mobile development

Μπορείς επίσης να γίνεις full stack developer ακολουθώντας περισσότερα από ένα career path (σύντομα διαθέσιμο).

ΤΙ ΧΡΕΙΑΖΟΜΑΙ?

Δεν είναι απαραίτητη καμία προηγούμενη γνώση προγραμματισμού. Όλες οι βασικές αρχές προγραμματισμού θα διδακτούν από την αρχή.

Από τεχνικό εξοπλισμό θα χρειαστεί να έχεις:

  • Έναν υπολογιστή για να ακολουθείς την ύλη
  • Ένωση στο Internet για να κατεβάσεις το software που απαιτείται για κάθε μάθημα

ΥΠΑΡΧΕΙ ΚΡΥΦΟ ΚΟΣΤΟΣ?

Δεν υπάρχει καμία απολύτως οικονομική υποχρέωση και όλα τα μαθήματα είναι εντελώς δωρεάν. Μπορείτε όμως να βοηθήσετε το site να μεγαλώσει με τους εξής τρόπους:

  • Κάνοντας ένα μικρό donation έτσι ώστε να μεγαλώσουν οι προσφερόμενες υπηρεσίες του site
  • Διαδίδοντας το link και σε άλλους ενδιαφερόμενους που θέλουν να μάθουν δωρεάν προγραμματισμό
  • Διαφημίζοντας την εταιρεία ή το προϊόν σας στο site
