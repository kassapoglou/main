Computed Properties

When you need to manipulate or calculate data before displaying it, use a computed property. Unlike standard methods, computed properties are cached. They only recalculate when their reactive dependencies change, saving massive amounts of processing power.

1. The Computed Object

Inside your Vue instance, you define a computed object containing functions. These functions must return a value.

computed: {
  totalAmmo() {
    return this.kinetic + this.plasma;
  }
}

2. Using it in the Template

You use computed properties in your HTML exactly like standard data variables (without parenthesis).

<p>Ammo Count: {{ totalAmmo }}</p>
