Breaking the Normal Flow

Normally, elements stack on top of each other. But sometimes, you need an element (like a badge, a close button, or a target reticle) to float exactly at specific coordinates. We use Absolute Positioning for this.

1. Ripping it from the flow

Applying position: absolute; tells the browser to ignore the element's normal place in line. It floats above everything else.

2. Establishing the Anchor

Absolute elements need a frame of reference to know where "top" and "left" actually are. They look up their family tree until they find a parent with position: relative;. That parent becomes the bounding box.

3. Setting the Coordinates

Once anchored, you steer the element using top, bottom, left, and right.

.target-drone {
  position: absolute;
  top: 10px;
  right: 10px;
}
