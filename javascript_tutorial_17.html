The Blueprint Factory

If you need to create 100 starships, writing 100 manual Objects is terrible coding. A Class is a blueprint for creating objects. The Constructor is a special method inside the class that runs automatically to "build" the object.

1. Defining a Class

Use the class keyword. Inside, add a constructor(). Use this to attach incoming parameters to the new object.

class Cruiser {
  constructor(name, hp) {
    this.name = name;
    this.hp = hp;
  }
}

2. Instantiation (The 'new' Keyword)

To actually build a ship using your blueprint, you instantiate it using the new keyword, passing in the specific data for that specific ship.

// We build ship 1
let shipOne = new Cruiser("Aegis", 500);
// We build ship 2
let shipTwo = new Cruiser("Titan", 800);
