Rendering Visual Data

Text is not enough. To display planetary scans, crew photos, or visual diagrams, we use the Image tag: <img>.

1. The Void Tag (Self-Closing)

Unlike headings and paragraphs, the image tag does not wrap around text. Therefore, it has no closing tag. It simply stands alone.

2. Essential Attributes: src & alt

The tag is useless without telling it what to display. We must use two attributes:

  • src (Source): The file path or URL of the image.
  • alt (Alternative Text): A description of the image in case the visual feed fails to load.
<!-- Displaying a visual feed -->
<img src="galaxy.png" alt="Spiral Galaxy">
