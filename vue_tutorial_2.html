Piping Data to the Screen

Your Vue instance holds the data, but the pilot needs to see it! To display a variable from your data() object directly into your HTML, we use Template Interpolation.

1. The Mustache Syntax {{ }}

By wrapping a variable name in double curly braces, Vue automatically replaces those braces with the actual value of the variable. If the data changes, the screen updates instantly.

<!-- If your Vue data has: { pilotName: "Viper" } -->

<p>Welcome aboard, {{ pilotName }}!</p>

<!-- Output: Welcome aboard, Viper! -->

2. Basic Expressions

You can also do basic math or JavaScript expressions directly inside the mustaches!

<!-- Assuming hull integrity is 50 -->
<span>{{ hullIntegrity * 2 }}%</span>
