Extending Blueprints

Inheritance allows a new class to take all the functionality of an existing class and build on top of it. This prevents code duplication and allows for specialized variations of objects.

1. The 'extends' Keyword

To inherit from a class, use the extends keyword when defining your new class.

class Bomber extends Starship { ... }

2. The 'super()' Function

If your specialized ship needs a new custom parameter in its constructor, you MUST call super() before using this. super() passes the base parameters up to the parent class's constructor to handle them.

class Bomber extends Starship {
  constructor(name, hull, payload) {
    super(name, hull); // Parent handles name & hull
    this.payload = payload; // Child handles the new trait
  }
}
