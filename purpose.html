Ο πρωταρχικός λόγος που με ώθησε στο να δημιουργήσω αυτή την ιστοσελίδα και να προσφέρω δωρεάν μαθήματα είναι γιατί πιστεύω ότι η γνώση, σε οποιοδήποτε επίπεδο, πρέπει να μεταδίδεται για να έχει αξία.

Όμως στην εποχή που ζούμε το να αποκτήσεις γνώση σημαίνει, τις περισσότερες φορές, ότι πρέπει να πληρώσεις ή γενικότερα να επενδύσεις, εκτός από τον χρόνο σου, και κάποιο χρηματικό ποσό για να αποκτήσεις αυτό που σου λείπει. Τουλάχιστον στον κόσμο του προγραμματισμού αυτό δεν θα έπρεπε να συμβαίνει.

Πιστεύω λοιπόν στα ακόλουθα και αυτοί είναι οι πρακτικοί λόγοι δημιουργίας του site:

1. Για την δική μου πρακτική εκμάθηση αλλά και για τα μαθήματα του site, χρησιμοποιώ γλώσσες προγραμματισμού που είναι δωρεάν, βάσεις δεδομένων που είναι δωρεάν, περιβάλλοντα ανάπτυξης εφαρμογών που είναι δωρεάν και δωρεάν application servers για να εκτελούνται οι εφαρμογές. Με αυτή την προσπάθεια λοιπόν προσφέρω και εγώ με τον δικό μου τρόπο πίσω στο open source community προσφέροντας δωρεάν μαθήματα προγραμματισμού.

2. Ξεκινώντας κάποιος τα πρώτα του βήματα στον κόσμο του προγραμματισμού, θα βρεθεί μπροστά σε πολλές επιλογές τόσο στις γλώσσες όσο και στις κατηγορίες. Για παράδειγμα, αν σε κάποιον αρέσει το Web τότε θα ακούσει για React, Angular, Vue.js, TypeScript κτλ, ενώ αν προτιμάει να γράφει τον κύριο κορμό της εφαρμογής (business logic) τότε Java, Python, C# είναι μερικές από τις γλώσσες που θα αντιμετωπίσει. Για να πειραματιστεί σε όλα αυτά μέχρι να βρει με τι ακριβώς θέλει να καταλήξει, θα έχει ξοδευτεί ένα σημαντικό χρηματικό ποσό. Ο σκοπός του site είναι να μάθει στον κάθε εκπαιδευόμενο τις βασικές αρχές προγραμματισμού σε πολλαπλές τεχνολογίες έτσι ώστε να αποφασίσει τι ακριβώς θέλει να κάνει χωρίς να πληρώσει κάποιο χρηματικό ποσό. Αφού το αποφασίσει, τότε θα μπορεί να επενδύσει σε αυτό και να παρακολουθήσει κάποια εξειδικευμένα σεμινάρια επάνω στην τεχνολογία που του/της αρέσει.

3. Υπάρχει ο φόβος και η λανθασμένη άποψη ότι ο προγραμματισμός είναι δύσκολος, και ότι δεν είναι κάτι προσιτό για όλους. Λάθος! Αν κάποιος καταλαβαίνει τα μαθήματα στο site μου, τότε καταλαβαίνει προγραμματισμό. “Είναι τόσο απλά?” θα αναρωτηθεί κάποιος. Ναι είναι. Απλά πρέπει να μπουν τα κουτάκια της γνώσης με την σωστή σειρά και την κατάλληλη στιγμή για να μπορέσει ο εκπαιδευόμενος να δει την αλληλουχία ανάμεσα τους. Και αυτό προσπαθώ να κάνω όσο το δυνατόν μπορώ μέσα από το site.

4. Μέσα από αυτό το site θέλω να βοηθήσω όλους όσους δεν έχουν οικονομική δυνατότητα, αλλά ούτε και πρόσβαση σε κάποια σεμινάρια για να επεκτείνουν τις γνώσεις τους. Για παράδειγμα απομακρυσμένα σημεία της Ελλάδας ή ακόμα κάποια νησιά. Αν υπάρχει Internet τότε αντί να ξοδεύουμε ώρες ατελείωτες στο facebook, μπορούμε να μάθουμε κάτι που μπορεί να μας οδηγήσει στο επόμενο βήμα της καριέρας μας.

5. Τελευταίο σημείο είναι η προσωπική ικανοποίηση ότι κάποιος διάβασε τις ενότητες και ίσως έπιασε μια εργασία γύρω από αυτό το αντικείμενο, έστω και σαν junior, για να κάνει τα πρώτα του βήματα.
