Key-Value Pairs

Lists use numbered indexes (0, 1, 2) to find items. But what if you want to look up an item by its name? In Python, we use Dictionaries. They are defined using curly braces {}.

1. Creating a Dictionary

Dictionaries store data in "Key": Value pairs. You provide the Key, and Python gives you the Value attached to it.

# A dictionary holding security clearance codes
passwords = {"Alpha": 1234, "Beta": 5678}

2. Extracting Data

To get data out of a dictionary, you do not use `.pop()` or an index number. Instead, you put the exact "Key" inside square brackets [].

# Get the value associated with the key "Beta"
my_code = passwords["Beta"]
print(my_code)
