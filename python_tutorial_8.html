Sensor Mapping

Dictionaries store data in Key:Value pairs. While lists use numbered indexes (like 0, 1, 2), dictionaries use descriptive Keys to rapidly locate data.

1. Creating & Accessing

Use curly braces {}. Connect keys and values with a colon :, and separate items with a comma.

data = {"hull": "stable", "energy": 100}

# Access data using the Key
print(data["hull"]) # Output: stable

2. Updating Data

Dictionaries are mutable. You can overwrite a value at any time.

# Change the energy value
data["energy"] = 90
print(data["energy"]) # Output: 90
