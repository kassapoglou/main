Grouping Data into Objects

Instead of creating ten different variables for one starship, we can group all its attributes into a single Object. An object stores data as key-value pairs.

1. Creating an Object

Objects are defined using curly braces {}. Inside, you define a key (property name), a colon, and its value.

let myShip = {
  name: "Valkyrie",
  shields: 100,
  isHostile: false
};

2. Dot Notation

To read or change a specific piece of data inside an object, use a dot (.) followed by the key name.

console.log(myShip.name); // Outputs: Valkyrie
myShip.shields = 50; // Updates shields
