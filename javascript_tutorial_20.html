Making the UI Reactive

Code that just runs top-to-bottom is rigid. Modern applications wait for the user to do something. An Event Listener is a sensor you attach to a DOM element. When a specific event happens (like a 'click'), it triggers a callback function.

The Syntax

First, grab the element. Then, use the addEventListener() method. It takes two arguments: the type of event (a string), and the function to run.

// 1. Select the target element
let btn = document.getElementById("launch-btn");

// 2. Attach the listener ('click' event)
btn.addEventListener('click', function() {
  // 3. This code ONLY runs when the button is clicked!
  console.log("Missiles Launched!");
});
