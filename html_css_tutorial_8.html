Painting the Hull (CSS)

HTML is just the structural titanium frame. To add colors, layouts, and energy shields, we use CSS (Cascading Style Sheets). There are three ways to inject CSS into your project.

1. Inline CSS (Direct Injection)

You can inject styles directly into a specific tag using the style attribute. Useful for quick, single-element fixes.

<h1 style="color: cyan;">Core Online</h1>
> RENDER OUTPUT: Core Online

2. Internal CSS (Sector Wide)

You can write rules inside a <style> tag located in the <head> of your document. This applies styles to the entire page at once.

<style>
  p {
    color: orange;
  }
</style>
> RENDER OUTPUT: All paragraphs on this page will now be orange.

3. External CSS (Fleet Wide)

The most powerful method. You create a separate `.css` file and link it using the <link> tag. This allows you to style thousands of pages from one control file!
