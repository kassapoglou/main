Hailing Frequencies (Input)

So far, we have been hardcoding data into our cargo boxes. But what if we want to ask someone else for information? We use Python's built-in input() function.

When the computer reads input(), it pauses the entire program, displays a message to the user, and waits for them to type an answer and press Enter.

# We ask the user for data, and store their answer in a variable.
color = input("Choose a laser color: ")
print(f"Lasers set to {color}!")
TERMINAL OUTPUT: Choose a laser color: Red > Lasers set to Red!

The Golden Rule of Input

The input() function always returns a String. Even if the user types the number 42, Python stores it as the text "42".
