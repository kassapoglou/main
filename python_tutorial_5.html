Talking to the Computer

To make your code interactive, you need to ask the user for commands. In Python, we use the input() function to open a communication channel.

1. Asking for Input

The input() function prints a question to the screen, pauses the game, and waits for the player to type an answer.

player = input("Enter pilot name: ")
print(f"Ready, {player}")
> Enter pilot name: Alex
> Ready, Alex

2. The Type Casting Trap

CRITICAL: Everything returned by input() is text (a String). If you ask for a laser power level, the computer sees the word "50", not the number 50. You must convert it into a real number using the int() function so your targeting computer can use it for math!

# Convert text to an integer number
power_text = input("Enter power: ")
laser_power = int(power_text)
