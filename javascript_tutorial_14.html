Processing the Fleet

To apply an action to every single item in an Array, we Iterate over it. Instead of writing identical code for index 0, index 1, and index 2, we use a loop to do the heavy lifting.

1. The Modern Way: for...of

The cleanest way to iterate an array is the for...of loop. It automatically creates a variable representing the *current* item and stops perfectly at the end of the array.

let fleet = ["Alpha", "Bravo", "Charlie"];

for (let ship of fleet) {
  console.log("Launching: " + ship);
}

2. The Classic Way: Array Length

You can also use a standard `for` loop, utilizing the array's .length property to tell the loop exactly when to stop, and accessing items via `array[i]`.

for (let i = 0; i < fleet.length; i++) {
  console.log(fleet[i]);
}
