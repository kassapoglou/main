Containers of Data

Variables are like labeled storage boxes. You create a box, give it a name, and put data inside it. We use the keyword let to declare a new variable.

1. Strings (Text)

Strings are sequences of characters wrapped in quotes (either single ' or double ").

let pilotName = "Maverick";

2. Numbers

Numbers are just digits. Do NOT put them in quotes, or JavaScript will treat them as Strings!

let shipLevel = 42;

3. Booleans

Booleans are the ultimate binary truth. They can only hold one of two values: true or false (no quotes).

let shieldsActive = true;
