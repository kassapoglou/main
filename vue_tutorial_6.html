Rendering Lists (v-for)

When your Vue instance contains an Array of data, you can use the v-for directive to automatically generate an HTML element for every item in that array.

1. The v-for Syntax

The syntax is item in items, where items is your array, and item is an alias you create for the current element being processed.

<li v-for="ship in fleet">
  {{ ship.name }}
</li>

2. The :key Attribute

Vue is highly efficient. When an array changes, it tries to patch existing DOM elements rather than redrawing everything. To help Vue track which elements are which, you must provide a unique :key for every looped element.

<div v-for="ship in fleet" :key="ship.id"></div>
