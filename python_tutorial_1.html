The Magic Cargo Boxes

In deep space, data is your most valuable resource. To use data, we must store it in Variables. Think of a variable as a magic cargo box with a label on the outside, and data on the inside.

1. Storing Different Cargo Types

To create a variable, type a name, use the equals sign =, and give it a value. Different types of data look different to the computer.

# A String (Text) MUST be wrapped in quotes
pilot_name = "Commander"
print(pilot_name)
> OUTPUT: Commander

# An Integer (Whole Number) has NO quotes
ship_shields = 100
print(ship_shields)
> OUTPUT: 100

2. The Naming Rules

The ship's computer is very strict about how you label your boxes. If you break these rules, the system will throw a Syntax Error.

  • No Spaces: Use underscores ( _ ) instead. This is called snake_case.
  • No Numbers at Start: A variable can contain numbers, but cannot begin with one.
# CORRECT NAMING:
cargo_type = "Food"
sector_4 = "Clear"

# INCORRECT NAMING (WILL CRASH!):
cargo type = "Food" # Has a space
1st_cargo = "Food" # Starts with a number
