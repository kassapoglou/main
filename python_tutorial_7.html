Processing the Queue

You know how to add things to a list. But how do you remove them to process them? In Python, we use the .pop() method to extract and remove an item from an array simultaneously.

1. Accessing by Index

Every item in a list has a numbered position, starting at 0. We pass this number into the `.pop()` method.

# Let's remove the first item (Index 0)
planets = ["Mars", "Venus", "Pluto"]
target = planets.pop(0)

print(f"Scanning: {target}")
> OUTPUT: Scanning: Mars

2. The List Updates Automatically

Because we used `.pop()`, the item is completely gone from the original list, and all remaining items shift forward!

print(planets)
> OUTPUT: ['Venus', 'Pluto']
