Understanding Data

Variables can hold different types of cargo. The computer treats text very differently than it treats numbers.

1. The Core Data Types

  • String (str): Text, wrapped in quotes. "Earth"
  • Integer (int): Whole numbers. 3
  • Float (float): Decimal numbers. 45.5
  • Boolean (bool): True or False. True

2. Formatting (f-strings)

To mix different data types into one message without crashing, Python uses f-strings. Place an f right before the opening quote, and put your variables inside curly braces {}.

name = "Commander"
level = 2

# Notice the 'f' before the quotes!
msg = f"Welcome {name}, you are level {level}."
print(msg)
> OUTPUT: Welcome Commander, you are level 2.
