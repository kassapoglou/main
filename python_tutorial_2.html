Data Signatures (Data Types)

Commander, cargo boxes don't just hold numbers. The ship's computer categorizes data into specific Data Types so it knows how to process them.

  • Strings (str): Used for text and communications. They MUST be wrapped in quotes (e.g., "Alpha" or 'Bravo').
  • Integers (int): Whole numbers used for counting, like crew members or missiles (e.g., 50).
  • Floats (float): Decimals used for precise measurements, like lightyears or frequencies (e.g., 88.5).

The f-string (Data Splicing)

Often, you need to combine text (Strings) with numbers (Integers/Floats) to send a transmission. If you try to mash them together normally, the system will crash.

The solution is an f-string (Formatted String). By putting the letter f right before your quotes, you can inject variables directly into the text using curly braces {}.

# Variable Assignments
pilot = "Maverick"
speed = 9.5

# The f-string formatting
log = f"Pilot {pilot} is at warp {speed}!"
print(log)
