MISSION BRIEFING

Handling DOM Events

To make your ship interactive, you must listen to user actions (like clicks). Vue uses the v-on directive to listen to DOM events and run JavaScript in response.

1. The v-on Syntax

Attach v-on:click to an element and point it to a method defined in your Vue instance.

<button v-on:click="fireWeapon">FIRE</button>

2. The @ Shorthand (Standard)

Because event listening is so common, Vue provides the @ shorthand. You will almost exclusively see this used in modern Vue fleets.

<!-- These do exactly the same thing: -->
<button v-on:click="fireWeapon">FIRE</button>
<button @click="fireWeapon">FIRE</button>
