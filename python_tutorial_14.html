Parameters vs. Arguments

To make a function truly reusable, we give it parameters (variables inside the parentheses). When we actually call the function, the specific data we hand to it are called arguments.

1. Multiple Parameters

A function can accept as many inputs as you need, separated by commas. The order in which you pass the arguments must match the order of the parameters!

def set_coordinates(x, y, z):
    print(f"Navigating to {x}, {y}, {z}")

# Passing 3 arguments to the 3 parameters
set_coordinates(10, 45, 90)
> OUTPUT: Navigating to 10, 45, 90

2. Mixing Data Types

Arguments can be any data type (Strings, Integers, Booleans). Just ensure you use quotes for Strings, but NOT for numbers!

def activate_shield(ship_name, power_percent):
    print(f"{ship_name} shields at {power_percent}%")

activate_shield("Apollo", 100)
> OUTPUT: Apollo shields at 100%
