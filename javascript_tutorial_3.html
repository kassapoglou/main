Arithmetic & The Math Object

To engage the hyperdrive and plot a course to the next planet, you must calculate precise telemetry using JavaScript's arithmetic operators and the built-in Math library.

1. Basic Operators

Calculate values using addition (+), subtraction (-), multiplication (*), and division (/).

let speed = 500 + 200; // 700
let distance = 1000 - 150; // 850

2. The Math Object

For advanced navigation, JavaScript has a built-in Math object filled with complex functions.

  • Math.round(4.6) rounds to nearest integer (5).
  • Math.pow(5, 2) calculates exponents (5 squared = 25).

3. Micro-Adjustments (Randomness)
