Saving Data to Disk

Variables are deleted when your script finishes. To save data permanently, we write it to a physical file. We do this using the open() function alongside the with keyword, which ensures the file safely closes when we are done.

1. Writing to a File

To create or overwrite a file, we open it in write mode by passing "w" as the second argument. Then we use the .write() method.

with open("captains_log.txt", "w") as file:
    file.write("Stardate 4590: All systems nominal.")
> [SYSTEM]: File 'captains_log.txt' created and saved to hard drive.

2. Reading from a File

To read a file that already exists, we open it in read mode using "r" and use the .read() method to extract the text.

with open("captains_log.txt", "r") as file:
    data = file.read()
    print(data)
