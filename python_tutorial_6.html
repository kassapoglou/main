Managing Inventory Arrays

Variables usually hold one piece of data. But what if a starship has 10,000 items in its cargo bay? In Python, we use Lists (also called arrays) to store multiple items in a single variable. Lists are defined using square brackets [].

1. Creating a List

Items in a list are separated by commas. Every item has a numbered position called an index, starting at 0.

# A list holding 3 strings
inventory = ["Blaster", "Medkit", "Comms"]
print(inventory)
> OUTPUT: ['Blaster', 'Medkit', 'Comms']

2. Appending Data

Lists are dynamic! You can add new cargo to the very end of an existing list using the .append() method.

# Add a new item to the bay
inventory.append("Shield_Core")
print(inventory)
> OUTPUT: ['Blaster', 'Medkit', 'Comms', 'Shield_Core']
