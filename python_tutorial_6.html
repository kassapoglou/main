The Cargo Array (Lists)

So far, every variable has been a single cargo box holding one item. But what if we need to store multiple items together? We use a List.

In Python, Lists are created using square brackets []. Inside the brackets, you separate your items using commas ,.

# A list holding three strings
weapons = ["laser", "emp", "torpedo"]
print(weapons)
