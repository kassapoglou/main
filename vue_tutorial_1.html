Bringing the Ship Online

Before you can command the DOM, you must create a Vue application instance and attach it to your HTML. This is the heart of every Vue project.

1. Vue.createApp()

This method generates a new application instance. You pass it an object containing your ship's data and logic.

const app = Vue.createApp({
  data() {
    return { status: 'Ready' }
  }
});

2. The .mount() Method

Creating the app isn't enough. You must plug it into the hull! The .mount() method connects your Vue instance to a specific HTML element via its ID.

app.mount('#app'); // Binds to <div id="app">
