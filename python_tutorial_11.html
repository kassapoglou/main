Continuous Logic (While)

An if statement checks a condition once. A while loop checks a condition and runs the code block repeatedly as long as the condition remains True.

The Danger of Infinite Loops

You MUST change the variable inside the loop so the condition eventually becomes False. If you don't, the loop runs forever, crashing the computer!

fuel = 0
while fuel < 3:
    print("Pumping...")
    # CRITICAL: Increase fuel to avoid infinite loop!
    fuel += 1
