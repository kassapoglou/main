Continuous Automation

Sometimes we need code to run repeatedly until a specific job is finished. A while loop will execute its block of code over and over again as long as its condition remains True.

1. The Loop Cycle

Every time the loop finishes its indented block, it jumps back to the top and checks the condition again.

fuel = 0
while fuel < 2:
    print(f"Fuel at {fuel}")
    fuel += 1 # Increase by 1
print("Tank Full!")

> OUTPUT:
> Fuel at 0
> Fuel at 1
> Tank Full!

⚠ CRITICAL: THE INFINITE LOOP

You must change the variable inside the loop! If we forgot to write fuel += 1 in the example above, the fuel would stay at 0 forever, the condition would always be True, and the computer would crash running the loop for eternity.
