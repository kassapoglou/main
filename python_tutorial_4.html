Hacking Communications

In Python, Strings (text data) come equipped with built-in tools called Methods. Methods allow you to instantly format, clean, or alter text without rewriting it manually.

You trigger a method by adding a dot . and the method name directly after a string variable.

signal = "hello fleet"
# .upper() converts everything to uppercase
print(signal.upper())
TERMINAL OUTPUT: > HELLO FLEET

Common String Methods

  • .upper() - Converts all letters to UPPERCASE.
  • .lower() - Converts all letters to lowercase.
  • .title() - Capitalizes the First Letter Of Each Word.
  • .replace("a", "b") - Swaps an old word for a new one.
msg = "WARNING: HULL BREACH"
fixed_msg = msg.lower()
print(fixed_msg)
TERMINAL OUTPUT: > warning: hull breach
