Molding the Data

Strings (text data) are rarely clean when intercepted. Python provides built-in methods to clean and format strings dynamically.

1. Cleaning Whitespace

Use .strip() to remove invisible spaces from the beginning and end of a string.

# Notice the extra spaces inside the quotes
raw_data = " access_code "
clean_data = raw_data.strip()
print(clean_data)
> OUTPUT: access_code

2. Changing Case

You can force a string to be all uppercase using .upper().

# You can chain methods together to do both at once!
password = " alpha_bravo "
final_code = password.strip().upper()
print(final_code)
> OUTPUT: ALPHA_BRAVO
