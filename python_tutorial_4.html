Molding the Data

Strings (text) are rarely clean when we receive them. Python provides built-in "methods" (tools attached directly to variables) to clean, slice, and manipulate strings dynamically.

1. Cleaning Whitespace

Data often arrives with accidental spaces. Use .strip() to remove spaces from the beginning and end.

# Notice the extra spaces inside the quotes
raw_data = " secret_code "
clean_data = raw_data.strip()
print(clean_data)
> OUTPUT: secret_code

2. Changing Case

You can force a string to be all uppercase or lowercase using .upper() or .lower().

password = "alpha_bravo"
shout = password.upper()
print(shout)
> OUTPUT: ALPHA_BRAVO
