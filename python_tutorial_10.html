Binary Branching

Programs become "intelligent" when they evaluate conditions and make decisions. We use if and else statements to split our code into two possible paths based on a Boolean (True/False) evaluation.

1. The 'If' Statement

Python checks the condition. If it evaluates to True, it runs the indented block of code.

systems_ready = True

if systems_ready:
    print("Launching Ship!")
> OUTPUT: Launching Ship!

2. The 'Else' Fallback

If the condition is False, Python skips the `if` block and jumps directly to the `else` block to execute a fallback plan.

fuel_level = 10 # Low fuel

if fuel_level > 50:
    print("Engage Thrusters")
else:
    print("Warning: Refuel needed")
> OUTPUT: Warning: Refuel needed
