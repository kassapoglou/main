Forking the Path (If/Else)

The ship's computer cannot just execute commands in a straight line. It must make decisions. We do this using if and else statements.

Syntax & Indentation

Python uses a colon : to start a decision block, and indentation (spaces) to know which code belongs inside that block.

# If condition is True, run this block:
if shield_power > 50:
    print("Shields stable.")
# Otherwise (False), run this block:
else:
    print("WARNING: SHIELDS LOW")
