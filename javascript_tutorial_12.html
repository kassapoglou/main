The Automation Loop

A for loop repeats a block of code a specific number of times. It is built using three distinct statements separated by semicolons.

for (Initialization; Condition; Increment) {
  // Code to execute repeatedly
}

1. The Three Stages

  • Initialization: Create a counter variable (usually let i = 0). This runs exactly once.
  • Condition: The loop runs as long as this is true (e.g., i < 5).
  • Increment: How to change the counter after every loop (e.g., i++ to add 1).

2. Putting it Together

This loop counts from 0 to 4, running the code exactly 5 times.

for (let i = 0; i < 5; i++) {
  console.log(`Charging array ${i}`);
}
