Responsive Adaptation

A layout designed for a large monitor will break on a mobile device. We use Media Queries to detect the user's screen width and apply specific CSS rules to adapt the interface.

1. The Trigger Condition

The @media rule checks conditions. (max-width: 600px) means: "Apply this CSS *only* if the screen is 600px wide or smaller."

2. Overriding the Layout

Inside the media query block, you rewrite CSS rules. A common tactic is changing a Flexbox container from a horizontal row to a vertical column so elements stack neatly on mobile screens.

@media (max-width: 600px) {
  /* Mobile specific rules here */
  .container {
    flex-direction: column;
  }
}
