Code Reusability

Instead of rewriting the same code to mine different asteroids, master programmers build Custom Functions. A function is a packaged block of code that you can trigger just by calling its name. We define them using def.

1. Defining and Calling

First you define the function (teach the computer the command). Later, you call the function to actually run it.

def fire_thrusters(): # 1. Defining it
    print("Vrooooom!")

fire_thrusters() # 2. Calling it
> OUTPUT: Vrooooom!

2. Parameters (Arguments)

Functions become incredibly powerful when they accept inputs, called parameters. You place a variable name inside the parentheses so the function knows what specific target to act upon.

def greet_pilot(name):
    print(f"Welcome to the bridge, {name}!")

greet_pilot("Commander")
> OUTPUT: Welcome to the bridge, Commander!
