Dynamic Cargo Arrays

Arrays are not static. You can load and unload data at will using built-in Array Methods. These are functions specifically attached to array variables.

1. Adding Items: push() & unshift()

.push(item) adds a new item to the END of the array.
.unshift(item) adds an item to the BEGINNING.

let cargo = ["Fuel"];
cargo.push("Ammo"); // ["Fuel", "Ammo"]

2. Removing Items: pop() & shift()

.pop() removes and returns the LAST item in the array.
.shift() removes and returns the FIRST item.

let cargo = ["Fuel", "Ammo"];
cargo.pop(); // Removes "Ammo", array is now ["Fuel"]

3. Array Length

You can always check how many items are currently in your array using the .length property.
