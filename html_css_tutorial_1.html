The Skeleton of the Web

Commander, to interface with the galactic network, you must understand its core language: HTML (HyperText Markup Language). HTML uses tags to define the structure of your data. Think of tags as containers that tell the browser what type of data is inside.

1. The Heading Tags

Tags usually come in pairs: an opening tag `<...>` and a closing tag ``. The <h1> tag creates the largest, most important text on the screen. (There are also smaller headings from h2 down to h6).

<h1>Galactic Cruiser</h1>
> RENDER OUTPUT: Galactic Cruiser

2. The Paragraph Tag

To add regular text or descriptions beneath your headings, we use the <p> tag.

<p>Pilot Name: Alex Vance</p>
> RENDER OUTPUT: Pilot Name: Alex Vance
