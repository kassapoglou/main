Iterating Through Data

While a while loop runs based on a True/False condition, a for loop is designed to automatically execute code once for every single item in a list or sequence.

1. Looping Over Lists

When you write for item in array:, Python pulls the first item out of the array, temporarily assigns it to the variable item, and runs the indented code. Then it automatically grabs the next one, and repeats!

squad = ["Alpha", "Beta"]
for pilot in squad:
    print(f"Deploying {pilot}")
> OUTPUT:
> Deploying Alpha
> Deploying Beta

2. Looping a Specific Number of Times

If you don't have a list, but just want to repeat an action exactly 3 times, you use the range() function.

for i in range(3):
    print("Warning Beep!")
> OUTPUT:
> Warning Beep!
> Warning Beep!
> Warning Beep!
