The 3rd Dimension (Z-Axis)

When elements overlap, the browser decides which one goes on top based on their order in the HTML. Elements written later usually sit on top. We can override this using the z-index property.

1. The Prerequisite

CRITICAL RULE: z-index ONLY works on elements that have a position applied (relative, absolute, fixed, or sticky). It will do nothing to static elements.

2. Pushing and Pulling

Think of z-index as physical elevation.
• Higher numbers (e.g., 10, 999) pull the element closer to your eyes.
• Negative numbers (e.g., -1) push the element deep into the background.

z-index: 1
z-index: 10
z-index: -1
