Single Source of Truth

Imagine building a dashboard with 50 elements that share the exact same neon green color. If the client wants to change it to blue, changing 50 lines of code is inefficient. We solve this with CSS Variables (Custom Properties).

1. The Generator (:root)

We define global variables inside the :root pseudo-class. This represents the absolute top level of the document. Variable names MUST start with two dashes (e.g., --main-color).

:root {
  --core-energy: #00ffcc;
}

2. Tapping the Energy (var())

To use the variable on an element, you wrap the variable name inside the var() function.

.btn {
  border-color: var(--core-energy);
}
