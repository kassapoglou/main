Reusable Logic Blocks

A function is a block of code designed to perform a specific task. Instead of writing the same lines of code repeatedly, you package them into a function and call it whenever needed.

1. Defining a Function

Use the function keyword, give it a name, and place the code inside curly braces {}.

function scanSector() {
  console.log("Scanning for anomalies...");
}

2. Parameters (Inputs)

Functions can accept data, called parameters, to make them dynamic. They act like variables that only exist inside the function.

function setCourse(destination) {
  console.log(`Plotting course to ${destination}.`);
}

3. Calling the Function

Defining a function doesn't run it. You must invoke (call) it by using its name followed by parentheses.

setCourse("Mars"); // Output: Plotting course to Mars.
