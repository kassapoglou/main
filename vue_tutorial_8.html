Event Modifiers

When handling DOM events, we often need to stop standard browser behaviors or contain "event bubbling" (where clicking an element also clicks its parent). Vue provides event modifiers as dot-suffixes to solve this declaratively.

1. The .prevent Modifier

Calls event.preventDefault(). It stops a link from navigating to a new URL, or a form from reloading the page upon submission.

<form @submit.prevent="launchMissile">

2. The .stop Modifier

Calls event.stopPropagation(). It prevents the click from traveling up the HTML tree to trigger parent elements.

<!-- You can chain them together! -->
<a href="/self-destruct" @click.prevent.stop="containmentField">
