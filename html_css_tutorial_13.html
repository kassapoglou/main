2D Grid Systems

Flexbox handles single rows perfectly. But when you need to layout elements in both rows and columns simultaneously, you need CSS Grid. It transforms a container into a powerful 2D matrix.

1. Activation & Columns

After applying display: grid;, use grid-template-columns to define the width of each column. A special unit, fr (fractional unit), distributes available space.

grid-template-columns: 1fr 1fr; (Two equal columns)
1fr
1fr

2. Spacing (The Gap)

Never use margins to space grid items. Instead, use the gap property on the parent to create perfect gutters between rows and columns.

.matrix {
  display: grid;
  grid-template-columns: 1fr 1fr;
  gap: 20px;
}
