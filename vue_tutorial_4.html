Conditional Rendering

Sometimes you only want an HTML element to exist if a specific condition is true. The v-if directive completely removes or adds an element to the DOM based on a boolean value.

1. The v-if Directive

Attach v-if to an element. If the data evaluates to true, the element renders. If false, it is destroyed.

<div v-if="shieldsActive">
  Shields are holding!
</div>

2. v-else-if and v-else

You can chain conditions exactly like standard JavaScript. A v-else element must immediately follow a v-if element.

<div v-if="shieldsActive">DEFLECTING</div>
<div v-else>HULL EXPOSED</div>
