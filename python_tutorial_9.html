The Power of Truth

In Python, a Boolean is a data type that can only have one of two values: True or False. (Note the capital letters!)

Logic Gates

We combine booleans using logical operators to make complex decisions:

  • and: BOTH sides must be True.
  • or: AT LEAST ONE side must be True.
  • not: Flips the value (True becomes False).
key_inserted = True
password_correct = False

# This evaluates to False because 'and' needs both to be True
access = key_inserted and password_correct
