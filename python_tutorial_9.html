True, False, and Logic Gates

Computers make decisions using Booleans. A boolean can only have one of two values: True or False (Notice the capital letters in Python!).

1. The AND Gate

We combine booleans using logic gates. The and operator requires both sides to be True for the final result to be True.

shields_up = True
weapons_hot = True
# 'safe_to_engage' will be True only if both are True
safe_to_engage = shields_up and weapons_hot

2. OR / NOT Gates

The or operator requires only one side to be True. The not operator flips True to False (and vice versa).

hull_breach = False
engines_offline = True
# Result is True because at least one is True
emergency = hull_breach or engines_offline
