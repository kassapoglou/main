The CSS Box Model

Every HTML element is essentially a rectangular box. To control layout and spacing, you must manipulate the three defensive layers wrapped around the central content.

1. Padding (Inner Shield)

Padding generates space inside the box, between the content (text) and the outer border. It prevents text from touching the walls.

padding: 20px;

2. Border (The Hull)

The border is the physical wall of the box. It wraps around the padding. You must define its thickness, style, and color.

border: 2px solid orange;
BORDER

3. Margin (Outer Forcefield)

Margin generates empty space outside the border. It acts like a repulsor field, pushing other external elements away from your box so they don't collide.

margin: 30px;
