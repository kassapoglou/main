The Switch Matrix

When evaluating a single variable against many possible exact values, a switch statement is much cleaner than a long chain of else if blocks.

1. Syntax & Cases

You pass the variable into the switch(). Then, you define case blocks for each possible match.

let cmd = "scan";

switch (cmd) {
  case "fire":
    console.log("Firing lasers!");
    break;
  case "scan":
    console.log("Scanning sector.");
    break;
}

2. The 'break' Keyword

CRITICAL: You MUST include a break; at the end of each case! If you forget it, the code will "fall through" and accidentally run the code in the next cases too!

3. The 'default' Fallback

Just like an else block, default: catches anything that didn't match any of your cases. It goes at the very end.
