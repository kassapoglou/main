Ordered Data Lists

An array is a special variable that can hold more than one value at a time. Think of it as a hangar bay with numbered parking spots.

1. Creating an Array

Arrays are created using square brackets []. Items inside are separated by commas.

let ships = ["Interceptor", "Bomber", "Frigate"];

2. Zero-Based Indexing

To access a specific item, you call the array name followed by the item's number in brackets. CRITICAL: Computers start counting at 0!

  • ships[0] is "Interceptor"
  • ships[1] is "Bomber"
  • ships[2] is "Frigate"

3. Extracting Data

You can pull an item out of an array and store it in a new variable, or log it directly.

let leadWing = ships[0];
console.log(leadWing); // Outputs: Interceptor
