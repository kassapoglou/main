MISSION BRIEFING

Continuous Execution

A while loop is simpler than a for loop. It only has one part: a condition. It will repeat its code block endlessly, as long as that condition remains true.

1. The Structure

If you don't know exactly how many hits it will take to destroy an enemy, you just tell the system to keep firing *while* their health is above 0.

let fuel = 10;
while (fuel > 0) {
  console.log("Engines running!");
  fuel -= 5; // Drain fuel
}

2. The Infinite Loop Danger!

If the condition never becomes false, the loop will run forever, freezing your entire computer system! You MUST alter the variable inside the loop.

// BAD: fuel never decreases. It will run forever!
while (fuel > 0) {
  console.log("Infinite burn!");
}
