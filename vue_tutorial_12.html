Dynamic Class Binding

You can dynamically toggle CSS classes based on your Vue data. Vue provides a special enhancement for class and style bindings, allowing you to pass Objects or Arrays.

1. The Object Syntax

By passing an object to :class, you can toggle multiple classes based on boolean data properties. The syntax is { 'css-class': condition }.

<!-- If isDamaged is true, the 'hull-critical' class is applied -->
<div :class="{ 'hull-critical': isDamaged }"></div>

2. Evaluating Expressions

The condition doesn't have to just be a boolean variable. It can be a full JavaScript expression determining which shield class to mount.

<div :class="{ 'shield-blue': weaponType === 'kinetic', 'shield-red': weaponType === 'thermal' }"></div>
