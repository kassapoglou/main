Branching Pathways

Programs must be able to make decisions. An if statement evaluates a condition (usually a Boolean). If the condition is true, the code inside its block runs. If it is false, it is skipped.

1. The 'If' Statement

The condition is placed inside parentheses (), and the execution code is placed inside curly braces {}.

let shieldsUp = true;
if (shieldsUp) {
  console.log("Safe from lasers.");
}

2. The 'Else' Fallback

What if the condition is false? You can chain an else block immediately after the `if` block to handle the alternative scenario.

let hasFuel = false;
if (hasFuel) {
  // Won't run
} else {
  console.error("Warning: Stranded.");
}
