Controlling Time

When a user hovers over an element, changing its color or size instantly feels aggressive and unpolished. We use the transition property to smoothly interpolate between the base state and the hover state.

The Anatomy of a Transition

The transition property is a shorthand that combines several rules:

  • Property: What to animate (e.g., background-color, or all for everything).
  • Duration: How long it takes (e.g., 0.5s or 300ms).
  • Timing Function: The pacing (e.g., ease, linear, ease-in-out).
/* ALWAYS put the transition on the BASE class, NOT the :hover class. */
.btn {
  transition: all 0.3s ease-in-out;
}
