Adding Behavior to Objects

Objects don't just hold static data (like strings and numbers). They can also hold functions! A function stored inside an object is called a Method. This allows objects to perform actions.

1. Creating a Method

You assign a function to a key just like you would a string.

let drone = {
  model: "Scout",
  scan: function() {
    console.log("Scanning sector...");
  }
};
drone.scan(); // Calls the method

2. The "this" Keyword

If a method needs to use data from its own object (like checking its own ammo), it must use the this keyword. this literally means "this exact object right here".

let ship = {
  energy: 100,
  fire: function() {
    this.energy -= 10; // Subtracts 10 from its own energy
  }
};
