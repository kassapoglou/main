Visual Modulation

Structural code is grey and boring. To make our interfaces readable and beautiful, we use CSS typography and color properties.

1. The color Property

Changes the color of the text. You can use standard names (like red, cyan), Hex codes (#ff5a00), or RGB values.

h2 { color: #00ffcc; }
> RENDER OUTPUT: Scanner Active

2. The font-family Property

Changes the typeface. You often list a primary font, followed by a generic backup (like sans-serif) in case the first one fails to load.

.terminal { font-family: 'Orbitron', sans-serif; }
> RENDER OUTPUT: TERMINAL FONT LOADED

3. The font-size Property

Scales the text. You can use pixels (px) or relative sizes like rem (which scales based on the root browser size).
