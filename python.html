Εδώ θα βρείτε μια ολοκληρωμένη σειρά δωρεάν μαθημάτων στην γλώσσα προγραμματισμού Python. Κάθε μια ενότητα είναι αυτόνομη από τις άλλες και δεν απαιτείται να γνωρίζετε τις προηγούμενες για να την παρακολουθήσετε. Επίσης σε κάθε ενότητα δημιουργούμε ένα καινούργιο πρόγραμμα Python έτσι ώστε να είναι εντελώς ξεκάθαρο το νόημα της ενότητας και η ουσία της θεωρίας που μελετάμε.

Αν ξεκινάτε τώρα την καριέρα σας σαν προγραμματιστές τότε να είσαστε σίγουροι ότι έχετε επιλέξει μια από τις πιο διάσημες γλώσσες προγραμματισμού που οι οπαδοί της αυξάνονται χρόνο με το χρόνο. Αυτό σημαίνει ότι θα μπορέσετε πολύ γρήγορα, με τα σημερινά δεδομένα αγοράς εργασίας, να απορροφηθείτε άμεσα σε κάποια εταιρεία πληροφορικής.

Αν γνωρίζετε Python μπορείτε πολύ εύκολα να κάνετε το επόμενο βήμα το οποίο είναι να γράφετε Python Web εφαρμογές με την χρήση frameworks όπως Django και Flask. Επίσης μπορείτε να προχωρήσετε στα δωρεάν μαθήματα Python Microservices όπου θα μπορείτε να δημιουργήσετε από τα πιο απλά μέχρι τα πιο σύνθετα Web Services και Microservices. Όποια και να είναι η κατεύθυνση σας είναι απαραίτητη προϋπόθεση να γνωρίζετε Python οπότε κάνατε την σωστή αρχή.
