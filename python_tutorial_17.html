Catching Crashes

When a script encounters a severe bug, it crashes the entire program. We can prevent this by using a Try/Except block. It tells the computer: "Try to do this, but if it breaks, don't crash—do this instead."

1. The 'try' Block

You place the dangerous or unstable code inside the try: block.

2. The 'except' Block

Immediately after, you place the except: block. This code only runs if the `try` block failed.

try:
    # This will cause an error (can't divide by zero)
    power = 100 / 0
except:
    # The crash is caught! Fallback activates.
    print("Error caught! Routing to backup battery.")
