Synthesizing Text

In order to communicate effectively, we must frequently combine variables (like a pilot's name) with static text. There are two primary ways to achieve this.

1. The Old Way (Concatenation)

Historically, we used the + operator to "add" strings together. This is highly prone to spacing errors and feels very clunky.

let msg = "Alert! " + pilot + " is in sector " + sector;

2. The Modern Way (Template Literals)

Template Literals allow you to embed variables directly inside the string without breaking it. Instead of regular quotes, you MUST use backticks ` ` (usually located above the Tab key).

To inject a variable, wrap it in ${ }.

let msg = `Alert! ${pilot} is in sector ${sector}.`;
