Toggling Visibility (v-show)

While v-if physically adds or removes an element from the DOM, v-show takes a different approach. It leaves the element in the HTML code, but toggles its inline CSS display property.

1. The v-show Directive

Attach v-show to an element. If the data is true, the element is visible. If false, Vue applies display: none.

<div v-show="isCloaked">
  Phantom Cloak Engaged.
</div>

2. v-if vs. v-show

Use v-if if the element rarely changes state (e.g., a login screen). Use v-show if you need to toggle something very frequently, like a cloaking device or a dropdown menu, because toggling CSS is much faster than rebuilding DOM nodes.
